Mockify is a Figma screen extractor tool. It connects to Figma, scans screens/flows/artboards, and generates professional mockups or interactive video snippets using AI.
MVP goal: Import a Figma file → generate downloadable mockups.
Later phases: Personal portfolio view with public/private toggle, global inspiration feed.
Tech stack: React + Vite, Tailwind CSS, shadcn components.
Design direction: Clean, modern, professional tool UI. Not opinionated on color yet.
When building, use shadcn components from @/components/ui/ for all UI elements.
